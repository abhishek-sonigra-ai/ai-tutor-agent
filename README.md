# AI Tutor Agent

This project builds an automated AI tutoring system.

Features

- generates weekly curriculum
- creates lessons for students
- sends assignments
- evaluates responses using AI
- produces feedback reports

Architecture

Schedule Trigger
↓
Topic Generator
↓
Lesson Generator
↓
Assignment Sender
↓
Student Submission
↓
AI Evaluation
↓
Feedback Report

Tools Used

- n8n
- LLM APIs
- prompt engineering
